# Tech Store Inventory Management System

## Overview

This project is a comprehensive inventory management system designed for a technology store that sells mobile phones, cases, headphones, chargers, and accessories. The application provides a modern administrative interface for managing products, stock, sales, and more, optimized for desktop and tablet use.

## Features

- **Dashboard**: View key metrics and insights about store performance.
- **Product Management**: Add, edit, and delete products with ease.
- **Stock Management**: Monitor current stock levels and manage stock adjustments.
- **IMEI/Serial Tracking**: Keep track of devices using unique identifiers.
- **Sales/POS**: Process transactions quickly with a user-friendly interface.
- **Stock Movement History**: Access a detailed history of stock movements and transactions.
- **Stock Alerts**: Receive notifications for low stock levels.

## Technology Stack

- **Frontend**: Next.js
- **Backend**: Supabase
- **UI Components**: shadcn/ui
- **Styling**: Tailwind CSS

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/tech-store-inventory.git
   ```
2. Navigate to the project directory:
   ```
   cd tech-store-inventory
   ```
3. Install dependencies:
   ```
   npm install
   ```
4. Set up your Supabase project and configure the environment variables as needed.

5. Run the development server:
   ```
   npm run dev
   ```

## PWA Configuration

This application is configured as a Progressive Web App (PWA). To install it on your device:

- Ensure you have the necessary icons and manifest file in the `public` directory.
- Access the application in a supported browser and follow the prompts to install.

## Usage

- Access the application through your browser at `http://localhost:3000`.
- Use the sidebar for navigation between different sections of the application.
- The sales screen features a quick search input and a clear cart display for efficient transaction processing.
